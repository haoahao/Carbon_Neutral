# Carbon Neutral Review Response Site

This repository hosts a static GitHub Pages site for:

- 114年度「碳中和中程計畫」審查意見修正對照表

## Safety Notes

- Pure static HTML (no JavaScript).
- No third-party scripts, trackers, or external assets.
- Data source is local `review_responses.tsv`.
- Site is generated by `build_site.py`.

## Update Content

1. Edit `review_responses.tsv`.
2. Run:
   ```bash
   python3 build_site.py
   ```
3. Commit and push generated files.
